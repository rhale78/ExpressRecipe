## Problem

The ProductService generated DAL classes failed to start with dependency injection errors:


**Root Cause:** The generated DAL classes require two dependencies that weren't registered in the DI container:
1. `IDbConnectionFactory` - For creating database connections
2. `RetryPolicyFactory` - For handling transient database errors with retry logic

## Solution

### 1. Used Existing HighSpeedDAL Aspire-Compatible Components

HighSpeedDAL already includes an Aspire-compatible SQL Server connection factory:

**File:** `src/HighSpeedDAL/src/HighSpeedDAL.SqlServer/SqlServerConnectionFactory.cs`

```csharp
public sealed class SqlServerConnectionFactory : IDbConnectionFactory
{
    private readonly ILogger<SqlServerConnectionFactory> _logger;

    public SqlServerConnectionFactory(ILogger<SqlServerConnectionFactory> logger)
    {
        _logger = logger ?? throw new ArgumentNullException(nameof(logger));
    }

    public async Task<IDbConnection> CreateConnectionAsync(
        string connectionString,
        DatabaseProvider provider,
        CancellationToken cancellationToken = default)
    {
        // Creates and opens a SQL Server connection
        SqlConnection connection = new SqlConnection(connectionString);
        await connection.OpenAsync(cancellationToken);
        return connection;
    }
}
```

### 2. Added Required Using Statements

### 3. Registered Dependencies in DI Container

**File:** `src/Services/ExpressRecipe.ProductService/Program.cs`

```csharp
// Register HighSpeedDAL components
// Register ProductDatabaseConnection for HighSpeedDAL
builder.Services.AddSingleton<ProductDatabaseConnection>();

// Register IDbConnectionFactory for HighSpeedDAL (required by generated DAL classes)
builder.Services.AddSingleton<IDbConnectionFactory, SqlServerConnectionFactory>();

// Register RetryPolicyFactory for HighSpeedDAL (required by generated DAL classes)
builder.Services.AddSingleton<RetryPolicyFactory>(sp =>
{
    var logger = sp.GetRequiredService<ILogger<RetryPolicyFactory>>();
    return new RetryPolicyFactory(
        logger,
        maxRetryAttempts: 3,
        delayMilliseconds: 100);
});

// Register InMemoryTableManager for high-speed in-memory operations with 30s flush to SQL
builder.Services.AddSingleton<InMemoryTableManager>(sp =>
{
    var logger = sp.GetRequiredService<ILogger<InMemoryTableManager>>();
    var connectionFactory = sp.GetRequiredService<ProductDatabaseConnection>();
    return new InMemoryTableManager(logger, () =>
    {
        var conn = new Microsoft.Data.SqlClient.SqlConnection(connectionFactory.ConnectionString);
        return conn;
    });
});

// Register generated DAL classes (auto-generated by HighSpeedDAL source generator)
builder.Services.AddScoped<ProductEntityDal>();
builder.Services.AddScoped<IngredientEntityDal>();
builder.Services.AddScoped<ProductImageEntityDal>();
builder.Services.AddScoped<ProductStagingEntityDal>();
builder.Services.AddScoped<ProductAllergenEntityDal>();
```
