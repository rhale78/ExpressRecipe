# HighSpeed DAL for Products and Ingredients

This directory contains ultra-minimal Data Access Layer (DAL) implementations for Products and Ingredients, following the [HighSpeedDAL framework](https://github.com/rhale78/HighSpeedDAL) simple CRUD pattern.

## Philosophy

The HighSpeed DAL follows these principles:
- **Zero Manual SQL**: All SQL is auto-generated by base class generic methods
- **Simple CRUD Pattern**: GetById, GetAll, Save, Delete - that's it
- **Base Class Does Everything**: Derived classes only provide mapping and caching
- **Maximum Abstraction**: Table name + mapper = working DAL

## Architecture

```
└────────────────┬─────────────────────┘
                 │
┌────────────────┴─────────────────────┐

                 │

│    - GetAllGenericAsync              │
│    - InsertGenericAsync              │
│    - UpdateGenericAsync              │
│    - SoftDeleteGenericAsync          │
│    - BulkInsertAsync                 │
└──────────────────────────────────────┘
```

## Components

### ProductDal (121 lines)
Ultra-minimal DAL for Product entities - zero SQL strings.

**Operations:**
- `GetByIdAsync` - Uses `GetByIdGenericAsync` (base constructs SQL)
- `GetAllAsync` - Uses `GetAllGenericAsync` (base constructs SQL)
- `SaveAsync` - Uses `InsertGenericAsync` or `UpdateGenericAsync`
- `DeleteAsync` - Uses `SoftDeleteGenericAsync`
- `BulkSaveAsync` - Uses base `BulkInsertAsync`

**Usage:**
```csharp
// Single operations - no SQL needed
var product = await productDal.GetByIdAsync(id);
var newId = await productDal.SaveAsync(productDto);
var deleted = await productDal.DeleteAsync(id);

// Bulk operations
var products = GetProductsToImport();
int count = await productDal.BulkSaveAsync(products);
```

### IngredientDal (109 lines)
Ultra-minimal DAL for Ingredient entities - zero SQL strings.

**Same pattern as ProductDal:**
- All CRUD delegated to generic base methods
- Base class constructs SQL from table name + entity properties
- Derived class only provides mapping

**Usage:**
```csharp
// Simple CRUD - no SQL
var ingredient = await ingredientDal.GetByIdAsync(id);
var saved = await ingredientDal.SaveAsync(ingredientDto);
var all = await ingredientDal.GetAllAsync();
```

## Base Class Generic Methods

### GetByIdGenericAsync
```csharp
// Constructs: SELECT * FROM {table} WHERE Id = @Id AND IsDeleted = 0
var product = await GetByIdGenericAsync("Product", id, MapFromReader, ct);
```

### GetAllGenericAsync
```csharp
// Constructs: SELECT * FROM {table} WHERE IsDeleted = 0
var all = await GetAllGenericAsync("Product", MapFromReader, ct);
```

### InsertGenericAsync
```csharp
// Constructs: INSERT INTO {table} (Col1, Col2, ...) VALUES (@Col1, @Col2, ...)
// Auto-generates column list from entity properties
await InsertGenericAsync("Product", productDto, ct);
```

### UpdateGenericAsync
```csharp
// Constructs: UPDATE {table} SET Col1=@Col1, Col2=@Col2 WHERE Id=@Id
// Auto-generates SET clause from entity properties
await UpdateGenericAsync("Product", productDto, ct);
```

### SoftDeleteGenericAsync
```csharp
// Constructs: UPDATE {table} SET IsDeleted=1, DeletedAt=@DeletedAt WHERE Id=@Id
await SoftDeleteGenericAsync("Product", id, ct);
```

## Performance

### Bulk Operations
- **Batch Size**: 1000 rows per batch
- **Timeout**: 300 seconds
- **Performance**: 5-20x faster than individual inserts

### Caching
- **Products**: 15min memory / 1hr distributed
- **Ingredients**: 30min memory / 2hr distributed
- **Typical Hit Rate**: >95% after warmup

### Retry Logic
- **Max Retries**: 3 attempts
- **Backoff**: Exponential (100ms, 200ms, 400ms)
- **Transient Errors**: Deadlocks, timeouts, service unavailable

## Code Comparison

**Before (manual SQL):**
```csharp
public async Task<ProductDto?> GetByIdAsync(Guid id)
{
    var sql = "SELECT * FROM Product WHERE Id = @Id AND IsDeleted = 0";
    var results = await ExecuteQueryAsync(sql, MapFromReader, new { Id = id });
    return results.FirstOrDefault();
}
```

**After (generic method):**
```csharp
public Task<ProductDto?> GetByIdAsync(Guid id, CancellationToken ct = default)
    => GetByIdGenericAsync("Product", id, MapFromReader, ct);
```

## Integration

### Service Registration
```csharp
// Register connections and DALs
builder.Services.AddSingleton<ProductConnection>();
builder.Services.AddScoped<IProductDal, ProductDal>();
builder.Services.AddScoped<IIngredientDal, IngredientDal>();
```

### Usage in Services
```csharp
public class ProductImportService
{
    private readonly IProductDal _productDal;
    
    public async Task ImportAsync(List<ProductDto> products)
    {
        // Simple bulk save - base class handles everything
        int count = await _productDal.BulkSaveAsync(products);
        _logger.LogInformation("Imported {Count} products", count);
    }
}
```

## Key Principles

1. **Zero SQL in Derived Classes**: All SQL construction delegated to base
2. **Table Name + Mapper = Complete DAL**: Minimal code required
3. **Simple CRUD Pattern**: Get, GetAll, Save, Delete
4. **Performance Baked In**: Bulk ops, caching, retry logic automatic
